TAURIWEBVIEW - ユーザーマニュアル (日本語)
================================================================================

クイックスタート
----------------
1. フォルダの内容をお好みの場所に展開します
2. このフォルダ内の "tauriwebview.exe" を探します
3. "tauriwebview.exe" をダブルクリックしてアプリケーションを起動します

必要要件
--------
- Windows 10以上
- インストール不要
- 管理者権限不要
- スタンドアロン実行ファイル（すべての依存関係が含まれています）

設定方法
--------
アプリケーションをカスタマイズするには、"config.ini" ファイルを編集します：

    [window]
    width = 1024          # ウィンドウの幅（ピクセル）
    height = 768          # ウィンドウの高さ（ピクセル）
    x = 100               # ウィンドウのX位置
    y = 100               # ウィンドウのY位置
    always_on_top = false # ウィンドウを常に最前面に表示（true/false）
    resizable = true      # ウィンドウのサイズ変更を許可（true/false）

編集後、変更を適用するためにアプリケーションを再起動してください。

含まれるファイル
----------------
- tauriwebview.exe     メインアプリケーション実行ファイル
- config.ini           アプリケーション設定ファイル
- contents/            ウェブアプリケーションリソース
  - index.html         HTMLファイル
  - main.js            JavaScriptコード
  - styles.css         スタイルシート
- README.txt           多言語ガイド
- README_EN.txt        英語マニュアル
- README_JA.txt        日本語マニュアル（このファイル）
- README_KO.txt        韓国語マニュアル

トラブルシューティング
----------------------
アプリケーションが起動しない場合：
  - すべてのファイルが同じフォルダにあることを確認してください
  - config.ini の文法が正しいことを確認してください
  - config.ini をデフォルト値にリセットしてみてください

ウィンドウが表示されない場合：
  - config.ini の x と y の値を確認してください
  - x=0、y=0 に設定して左上に配置してみてください

アプリケーションが遅い場合：
  - 他のアプリケーションを閉じてメモリを解放してください
  - インターネット接続を確認してください（該当する場合）
