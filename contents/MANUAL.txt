TAURIWEBVIEW - USER MANUAL

[ENGLISH]
--------

INSTALLATION & USAGE:
1. Extract the folder contents to a location of your choice
2. Locate "tauriwebview.exe" in the dist folder
3. Double-click on "tauriwebview.exe" to launch the application

REQUIREMENTS:
- Windows 10 or higher
- No installation required
- No administrator privileges required
- Standalone executable (all dependencies included)

CONFIGURATION:
- Edit "config.ini" to customize application behavior
- Available settings:
  - width: window width in pixels
  - height: window height in pixels
  - x: window x position
  - y: window y position
  - always_on_top: keep window on top (true/false)
  - resizable: allow window resizing (true/false)

FILES INCLUDED:
- tauriwebview.exe: Main application executable
- config.ini: Configuration file
- contents/: Application web resources (HTML, CSS, JavaScript)

TROUBLESHOOTING:
- If the app doesn't start, ensure all files are in the same directory
- Check that config.ini is properly formatted
- For issues, review the config.ini file for syntax errors

[日本語]
--------

インストール と 使用方法:
1. フォルダの内容を任意の場所に展開します
2. distフォルダ内の "tauriwebview.exe" を探します
3. "tauriwebview.exe" をダブルクリックしてアプリケーションを起動します

必要要件:
- Windows 10以上
- インストール不要
- 管理者権限不要
- スタンドアロン実行ファイル（すべての依存関係が含まれています）

設定:
- "config.ini" を編集してアプリケーション動作をカスタマイズします
- 利用可能な設定:
  - width: ウィンドウの幅（ピクセル）
  - height: ウィンドウの高さ（ピクセル）
  - x: ウィンドウのx位置
  - y: ウィンドウのy位置
  - always_on_top: ウィンドウを常に最前面に表示（true/false）
  - resizable: ウィンドウのサイズ変更を許可（true/false）

含まれるファイル:
- tauriwebview.exe: メインアプリケーション実行ファイル
- config.ini: 設定ファイル
- contents/: アプリケーションウェブリソース（HTML、CSS、JavaScript）

トラブルシューティング:
- アプリが起動しない場合、すべてのファイルが同じディレクトリにあることを確認してください
- config.ini が正しくフォーマットされていることを確認してください
- 問題が発生した場合は、config.ini ファイルの構文エラーを確認してください

[한국어]
--------

설치 및 사용 방법:
1. 폴더 내용을 원하는 위치에 압축 해제합니다
2. dist 폴더에서 "tauriwebview.exe"를 찾습니다
3. "tauriwebview.exe"를 더블클릭하여 애플리케이션을 실행합니다

요구사항:
- Windows 10 이상
- 설치 필요 없음
- 관리자 권한 필요 없음
- 독립 실행형 실행 파일 (모든 의존성 포함)

설정:
- "config.ini"를 편집하여 애플리케이션 동작을 사용자화합니다
- 사용 가능한 설정:
  - width: 창의 너비 (픽셀)
  - height: 창의 높이 (픽셀)
  - x: 창의 x 위치
  - y: 창의 y 위치
  - always_on_top: 창을 항상 앞에 유지 (true/false)
  - resizable: 창 크기 조절 허용 (true/false)

포함된 파일:
- tauriwebview.exe: 메인 애플리케이션 실행 파일
- config.ini: 설정 파일
- contents/: 애플리케이션 웹 리소스 (HTML, CSS, JavaScript)

문제 해결:
- 앱이 시작되지 않으면 모든 파일이 같은 디렉토리에 있는지 확인하십시오
- config.ini가 올바르게 포맷되어 있는지 확인하십시오
- 문제가 발생한 경우 config.ini 파일의 문법 오류를 확인하십시오
