================================================================================
TAURIWEBVIEW - 사용자 매뉴얼 (한국어)
================================================================================

빠른 시작
---------
1. 폴더 내용을 원하는 위치에 압축 해제합니다
2. 이 폴더에서 "tauriwebview.exe"를 찾습니다
3. "tauriwebview.exe"를 더블클릭하여 애플리케이션을 실행합니다

시스템 요구사항
---------------
- Windows 10 이상
- 설치 불필요
- 관리자 권한 불필요
- 독립 실행형 실행 파일 (모든 의존성 포함)

설정 방법
---------
애플리케이션을 사용자화하려면 "config.ini" 파일을 편집합니다:

    [window]
    width = 1024          # 창의 너비 (픽셀)
    height = 768          # 창의 높이 (픽셀)
    x = 100               # 창의 X 위치
    y = 100               # 창의 Y 위치
    always_on_top = false # 창을 항상 앞에 유지 (true/false)
    resizable = true      # 창 크기 조절 허용 (true/false)

편집 후 변경 사항을 적용하려면 애플리케이션을 다시 시작하세요.

포함된 파일
-----------
- tauriwebview.exe     메인 애플리케이션 실행 파일
- config.ini           애플리케이션 설정 파일
- contents/            웹 애플리케이션 리소스
  - index.html         HTML 파일
  - main.js            JavaScript 코드
  - styles.css         스타일시트
- README.txt           다국어 가이드
- README_EN.txt        영어 매뉴얼
- README_JA.txt        일본어 매뉴얼
- README_KO.txt        한국어 매뉴얼 (이 파일)

문제 해결
---------
애플리케이션이 시작되지 않는 경우:
  - 모든 파일이 같은 폴더에 있는지 확인하세요
  - config.ini 파일의 문법이 올바른지 확인하세요
  - config.ini를 기본값으로 재설정해 보세요

창이 나타나지 않는 경우:
  - config.ini의 x와 y 값을 확인하세요
  - x=0, y=0으로 설정하여 좌상단에 배치해 보세요

애플리케이션이 느린 경우:
  - 다른 애플리케이션을 종료하여 메모리를 확보하세요
  - 인터넷 연결을 확인하세요 (해당하는 경우)

지원
----
자세한 내용은 다른 언어의 README 파일을 참조하세요:

- README_JA.txt 일본어판
