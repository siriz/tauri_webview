ありがとうございます！
----------------------
TauriWebviewをダウンロードいただきありがとうございます！

このアプリケーションは、html/フォルダ内のHTML/CSS/JavaScriptファイルを
デスクトップアプリケーションとして表示します。


================================================================================
システム要件
================================================================================

✓ Windows 10以上
✓ 管理者権限は不要です
✓ 外部ソフトウェアのインストールは不要です


================================================================================
使用方法
================================================================================

1. 基本的な使用
--------------

  1) tauriwebview.exeをダブルクリックして実行
  2) アプリケーションが自動的にhttp://localhost:8000でWebサーバーを起動します
  3) html/フォルダの内容がウィンドウに表示されます


2. コンテンツの修正
------------------

  html/フォルダ内のファイルを自由に修正できます:

  • index.html - メインページ
  • styles.css - スタイルシート
  • main.js - JavaScriptコード
  • 画像、音声、動画、PDFファイルも追加可能

  [ショートカットキー]
  • F5: ページ更新（修正されたHTML/CSS/JSを反映）
  • F11: フルスクリーン切り替え（フルスクリーン ↔ ウィンドウモード）


3. 設定の変更 (config.ini)
--------------------------

  config.iniファイルをメモ帳で開いて設定を変更できます:

  [window]
  width=800              # ウィンドウ幅
  height=600             # ウィンドウ高さ
  x=100                  # ウィンドウX位置（オプション）
  y=100                  # ウィンドウY位置（オプション）
  always_on_top=false    # 常に前面表示 (true/false)
  resizable=true         # サイズ変更可能 (true/false)

  [app]
  port=8000              # Webサーバーポート（競合する場合は変更）

  [重要] 設定変更後は、アプリケーションを再起動する必要があります。


================================================================================
サポートされているファイル形式
================================================================================

html/フォルダに以下のファイルを追加して使用できます:

Webファイル:
  HTML, CSS, JavaScript, JSON

画像:
  PNG, JPG, JPEG, GIF, SVG, WebP, BMP, ICO

音声:
  MP3, WAV, OGG, M4A, AAC, FLAC

動画:
  MP4, WebM, AVI, MOV, MKV

ドキュメント:
  PDF, TXT, XML

フォント:
  WOFF, WOFF2, TTF, OTF


================================================================================
トラブルシューティング
================================================================================

Q: アプリケーションが起動しません
A: • ポート競合: config.iniのport値を別の番号に変更してみてください
     (例: 8080, 8888)
   • ファイアウォール: Windowsファイアウォールでアプリケーションを許可してください

Q: ファイルを修正しても反映されません
A: 1) アプリケーションウィンドウでF5キーを押して更新
   2) それでも反映されない場合は、アプリケーションを再起動

Q: ページが表示されません
A: • html/フォルダにindex.htmlファイルがあることを確認してください
   • ブラウザ開発者ツールでエラーを確認してください（F12キー）
